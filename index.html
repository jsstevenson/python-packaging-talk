Wheel? Egg?

  • "Precompiled" (faster to install)
  • Eggs came first
    • Nonstandardized structure, not versioned
    • Basically smashes a bunch of metadata files into other metadata files
  • Wheels are nicer. Standardized, versioned
    • Built with specific... specifications, in a virtual environment

sdist

  • Raw source file, in a tarball
  • Can define specific build steps
  • Relatively rare these days if you're on a Mac
